## O objetivo do trabalho:

  Este projeto tem como objetivo analisar dados de consumo energético em um ambiente de trabalho e propor soluções sustentáveis que contribuam para a redução de desperdícios e otimização do uso de energia.
  A ideia central é entender como fatores como ocupação, temperatura e rotina diária influenciam o consumo e, a partir disso, apontar caminhos reais para tornar o ambiente mais eficiente e preparado para o futuro.

---

## O que foi feito:

  Foi realizado uma análise utilizando um conjunto de dados simulados contendo informações por hora ao longo de um dia. Esses dados incluem consumo energético, temperatura ambiente e presença de ocupação no espaço. A partir deles, foi possível identificar padrões claros, como horários de pico, momentos de desperdício (consumo sem presença de pessoas) e tendência de uso ao longo do tempo.
  Os gráficos gerados permitem visualizar de maneira intuitiva como o consumo se comporta e como pequenas variações na ocupação influenciam a energia utilizada.

  Um ponto importante da análise foi detectar horas com consumo acima do esperado mesmo quando não havia ninguém no ambiente. Esse tipo de diagnóstico indica que equipamentos permanecem ligados sem necessidade, representando um desperdício direto e fácil de corrigir.
  Além disso, a regressão linear aplicada mostra a tendência geral do consumo ao longo do dia, evidenciando momentos críticos onde intervenções tecnológicas podem gerar bons resultados.

---

## Propostas de solução:

  Com base nos dados observados, algumas soluções foram propostas para reduzir o consumo e aumentar a sustentabilidade do ambiente.
  * A primeira delas é a implementação de automação para desligar equipamentos automaticamente em períodos sem ocupação.
   * Mesmo um sistema simples, baseado em sensores ou horários predefinidos, já reduziria boa parte do desperdício detectado.
  * utra solução envolve reorganizar atividades ou processos internos, direcionando tarefas de maior demanda energética para horários de menor carga, suavizando os picos de consumo.
Além dessas ações, também é possível integrar essa análise em uma rotina contínua de monitoramento, criando um painel de acompanhamento ou alertas automáticos para quando o consumo ultrapassar limites definidos.

---

## Concluindo: 

  Por fim, este trabalho reforça como a análise de dados pode ser uma ferramenta poderosa para transformar ambientes de trabalho, permitindo decisões mais inteligentes, econômicas e alinhadas com a sustentabilidade. Entender o consumo é o primeiro passo para melhorar, e este projeto permite a visualização desse caminho.
